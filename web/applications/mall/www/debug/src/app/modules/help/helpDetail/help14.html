售后服务退换货政策

  • 为了保障您的利益，请在收到货物后仔细清点并检查商品，如发现商品配送有误、包装破损、产品存在质量问题或是数量不对等情况，请在签收前当面向配送人员提出，或是及时拨打我们售后服务热线400-8199-491进行反馈，以便我们能尽快地为您处理。为保障您的权益，请仔细阅读以下内容:

    一、符合以下情况，可享受大厨网退换货政策

    商品签收前，您需认真核对收货确认单中商品（含赠品）的种类、数量和质量。如发现与下单商品不符或商品（不含赠品）包装破损等问题，您可以当场提出退换货要求。

    二、出现以下情况，大厨网将不支持退换货

    1. 生鲜类商品（水果、蔬菜、鲜肉禽蛋、海鲜水产、冷藏冻品等）当场验收，签收后将不享受退换货服务。

    2. 非生鲜类货品（米面粮油，调料干货，厨房用品等）在签收商品24小时后将不再享受退换货政策

    3. 非大厨网的商品，不予处理。

    4. 因您使用或保管不当导致出现包装破损或质量问题的商品，将不予受理。

    5. 签收后，由于您下错单造成的损失，将不享受退换货服务。

    6. 线上商品的图片及描述信息仅供参考，您收到的商品均以实物为准。因拍摄环境及显示屏色差等问题，可能造成商品的图片与实物有一定差距，不属质量问题，不在退换货范围之内。

    三、特别说明

    1. 为保障您权益，请您签收前与配送人员当面核对货品（含赠品）是否与订单一致，确认无误后再签收。

    2. 如果您的订单交由他人签收，签收人享有与您等同的权利，我们将视为您本人签收。

    3. 退换货时，您必须出具：退货商品/收货确认单和相关票据(请妥善保存您的收货单据)/问题商品相关照片等。

    大厨网全国统一售后服务热线：400-8199-491
