新手指南下单流程

  • 1,“登陆”个人帐户，欢乐的购物从这里开始。

    2,浏览产品分类，并选择商品，填好“数量”、“加入购物车”，能轻松选定您想要的。

    3,进入购物车进行结帐，一键购入我们为您优选的高品质、最新鲜的健康食材。

    4,核对订货数量及商品，提交“结算”。

    5,选择“送货时间”及“支付方式”，核对“收货信息”，并“提交订单”。

    6,核对配送时间，确认 。

    7,订单创建成功 。

    8,若选择“微信支付”，则扫描二维码，进行微信支付，输入密码，支付成功，静候大厨网在第一时间为您送达。

    9,如何查看订单信息？

    “登录账号”—— 进入“个人中心”——点击“我的订单”即可查看订单信息
