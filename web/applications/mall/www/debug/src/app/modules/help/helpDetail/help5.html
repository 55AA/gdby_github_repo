新手指南常见问题

  • Q:如何取消订单？

    您可以在我的订单中自行取消订单。如果无法取消，请拨打400-8199-491客服电话进行咨询，由工作人员帮您操作。

  • Q:如何修改配送时间？

    您可拨打400-8199-491客服电话或是通过微信公众号来申请修改配送时间。

  • Q:如何查询物流信息？

    您可拨打400-8199-491查询物流信息。

  • Q:收到的商品少了/发错货了怎么办？

    同个订单购买蔬菜，水果，可能不会同时送达，在配送时间内建议您耐心等待，如超过配送时间仍未收到，可直接拨打400电话联系我们。并且在签收货物时，请您核对商品种类、重量、质量、数量、规格、赠品等是否与订单一致，确认无误后再签收货物

  • Q:我能指定某一个司机送货吗？

    大厨网下单时不支持指定送货司机。
